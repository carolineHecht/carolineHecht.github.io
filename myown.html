FreedomPay testing

  • Register through the production ILR website, and then when you get into the registration system, e.g. when you're on the "Confirm Registration and Pay" screen, change the URL from "register.ilr.cornell.edu" to "registerdev.ilr.cornell.edu", re-enter your username and password, and you should be where you left off, but with a yellow background. Then you can click the Pay Now button and use the test credit card info below.

    Alternate way of looking at it: SDC register on registerdev: http://registerdev.ilr.cornell.edu/registration/account_home.htm
    (You need to have something in your cart already, and if you're not on a local machine, you might also need to add registerdev.ilr.cornell.edu as IP 128.253.61.72 in the hosts file, which is Windows\System32\drivers\etc\hosts - As of 12/2016 Michael Radzik added that to the ILR DNS, so I don't need to add it to my Windows hosts file, but I do need it on my Mac.)

  • FreedomPay wiki: https://freedompay.atlassian.net/ (Login is cornell.dev@freedompay.com / Cornell101.)
  • Finding FreedomPay test credit info as of 11/30/2016 (scroll down, not right/left arrows):
    1. Log in to the FreedomPay wiki site (see above)
    2. Click on "Hosted Payment Page (URL is https://freedompay.atlassian.net/wiki/display/EIT/Hosted+Payment+Page)
    3. Scroll down to "Certification" and click on the HPP Certification Plan - HPP CER Template.xlsm link.
    4. Then scroll down a little to find the info below.

Miscellaneous Information

  • To open a link in a new window every time use target="_blank".
  • Credit card testing: visa, 5000300020003003 (we used this with Elavon, not sure if it works with FreedomPay)
  • To selectively delete entries that have been saved for auto-filling a form field, use arrow keys to select and then press shift+delete. This works in Firefox, not sure about other browsers.
  • Here's a sample .htaccess file to allow include files in that dir; it just has a single line: 
    options includes
    and this is what you need to be able to get an index listing: 
    options Indexes
    For more info see the apache doc, options section.
